# Ecosystem

## Starter Kits

* [Zap.ts](https://github.com/zap-studio/monorepo) — Next.js boilerplate
* [Better-T-Stack](https://github.com/AmanVarshney01/create-better-t-stack) — CLI for end-to-end type-safe TypeScript projects
* [Vazen](https://github.com/stack-found/vazen) — Production-grade monorepo starter
* [create-start-app](https://github.com/TanStack/create-tsrouter-app) — React + TanStack Router + oRPC
* [create-o3-app](https://github.com/Tony-ArtZ/create-o3-app) — Bleeding-edge full-stack TypeScript stack
* [RT Stack](https://github.com/nktnet1/rt-stack) — Lightweight fullstack turborepo
* [Start UI](https://github.com/BearStudio/start-ui-web) — Opinionated TanStack Start starter
* [ShipFullStack](https://github.com/sunshineLixun/ShipFullStack) — React + TanStack Start + Hono + oRPC + Expo
* [WXT Starter](https://github.com/mefengl/wxt-starter) — Best template for wxt
* [Start Kit](https://github.com/CarlosZiegler/start-kit.dev) — Production-ready SaaS starter

## Tools

* [orpc-file-based-router](https://github.com/zeeeeby/orpc-file-based-router) — File-based router for oRPC
* [Vertrag](https://github.com/Quatton/vertrag) — Spec-first API development tool
* [Prisma oRPC Generator](https://github.com/omar-dulaimi/prisma-orpc-generator) — Generate routers from Prisma schema
* [DRZL](https://github.com/use-drzl/drzl) — Codegen for Drizzle ORM
* [orpc-msw](https://github.com/DanSnow/orpc-msw) — MSW mocking for OpenAPI oRPC contracts

## Libraries

* [Permix](https://permix.letstri.dev/) — Permissions management
* [trpc-cli](https://github.com/mmkal/trpc-cli) — Turn a router into a type-safe CLI
* [@reliverse/rempts](https://github.com/reliverse/rempts) — CLI toolkit
* [oRPC Shield](https://github.com/omar-dulaimi/orpc-shield) — Type-safe authorization
* [Every Plugin](https://github.com/near-everything/every-plugin) — Composable plugin runtime
* [orpc-json-diff](https://github.com/NeilTheFisher/orpc-json-diff) — JSON patches for event iterators
* [effect-orpc](https://github.com/utopyin/effect-orpc) — Effect-TS integration
